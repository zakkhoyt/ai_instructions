How to only show scanned files inside the view

By default the GrandPerspective views only show the files that have been scanned. The application can however be configured to show the entire volume. With this setting enabled the view also shows the free space and the miscellaneous used space. These are shown as respectively a black and grey rectangle. This way of viewing the data is mostly useful if you scanned the entire volume, as otherwise the miscellaneous used space can be very large.

If you want to only show the scanned files, you can disable "Show entire volume" from the view's Display tab. This then changes how the current view is displayed. To change the default mode for all views, you can configure the setting from the preferences.
